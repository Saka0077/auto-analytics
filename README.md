# Анализ объявлений авто

Веб-приложение для импорта объявлений, рейтинга машин, сравнения, избранного, истории и аккаунтов пользователей.

## Локальный запуск

```powershell
npm install
npm start
```

После запуска открой `http://localhost:3000`

## Что умеет

- импорт объявлений с `kolesa.kz` через сервер;
- фильтрация и рейтинг по цене, году, пробегу и владельцам;
- карточка объявления с объяснением рейтинга;
- сравнение 2-3 машин;
- экспорт сравнения в `CSV` и печать в `PDF`;
- аккаунты: регистрация, вход, выход;
- профили внутри аккаунта;
- избранное и история сравнений для пользователя.

## Основные файлы

- `index.html` - интерфейс
- `styles.css` - стили
- `app.js` - клиентская логика
- `server.js` - API и сервер
- `package.json` - команды запуска
- `.gitignore` - исключения для Git

## Деплой на Render

1. Создай новый репозиторий на GitHub.
2. Залей туда этот проект.
3. На Render создай `Web Service`.
4. Подключи GitHub-репозиторий.
5. Укажи настройки:

```text
Build Command: npm install
Start Command: npm start
```

6. Render сам выдаст публичный URL.

## Важно

- Папка `data/` не должна уходить в Git, поэтому она добавлена в `.gitignore`.
- На хостинге данные в файлах могут быть временными. Если нужен надёжный продакшн, следующим шагом лучше вынести пользователей и состояние в базу данных.
- Парсер `kolesa.kz` зависит от структуры HTML сайта. Если сайт изменится, парсер нужно будет поправить.
